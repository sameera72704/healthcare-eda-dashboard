# Heart Disease Analytics & Prediction Dashboard

## Overview

An interactive healthcare dashboard built using Python, Streamlit, Plotly, Pandas, and Scikit-learn.

This project combines data analysis, visualization, and machine learning to analyze clinical patient data and predict heart disease risk.

---

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Analytics Dashboard
- Machine Learning Risk Prediction
- User Input Prediction Interface
- Risk Probability Visualization

---

## Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
- Scikit-learn
- Git / GitHub

---

## Dataset

UCI Heart Disease Dataset

---

## Machine Learning Model

Model Used:

- Logistic Regression

Prediction Inputs:

- Age
- Sex
- Chest Pain Type
- Cholesterol
- Maximum Heart Rate

Output:

- Heart Disease Risk Prediction
- Risk Probability Score

---

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
